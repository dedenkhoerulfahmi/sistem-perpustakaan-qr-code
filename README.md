Deskripsi Proyek

Sistem Informasi Perpustakaan ini merupakan hasil dari kegiatan Kerja Praktik yang dilakukan oleh Deden Khoerul Fahmi dan Kusmawan. Tujuan dari proyek ini adalah untuk meningkatkan efisiensi dan digitalisasi proses pengelolaan perpustakaan, khususnya pada pencatatan buku, anggota, peminjaman, pengembalian, absensi pengunjung, serta penyusunan laporan.

Aplikasi ini dirancang dengan tampilan modern, mudah dipahami, dan responsif. Sistem dibangun menggunakan framework CodeIgniter 4, dikombinasikan dengan Bootstrap 5 dan template admin Modernize. Sistem ini tidak menggunakan fitur QR Code, seluruh proses dilakukan melalui input dan pengelolaan data secara manual melalui antarmuka web.
